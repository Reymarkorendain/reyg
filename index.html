What is CSS?

CSS is the language for describing the presentation of Web pages, including colors, layout, and fonts. It allows one to adapt the presentation to different types of devices, such as large screens, small screens, or printers. CSS is independent of HTML and can be used with any XML-based markup language.

Basic SYNTAX

Selector syntax This is used to select an HTML element on which you want to apply a style. It typically starts with the element name, followed by a set of curly braces containing the style properties and their values. For example:

Class syntax This is used to apply a style to elements with a specific class attribute. It starts with a period followed by the class name.

ID syntax This is used to apply a style to an element with a specific ID attribute. It starts with a pound sign followed by the ID name.

Pseudo-class syntax This is used to define a style for a specific state of an element, such as when it is being hovered over or clicked. It starts with a colon followed by the state name.

Media query syntax This is used to define styles that only apply when the device meets certain conditions, such as screen width. It starts with the @media keyword followed by the condition and the styles to apply.
